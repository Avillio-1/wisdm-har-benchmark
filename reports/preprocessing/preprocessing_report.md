# Preprocessing Report

Selected stream: `phone` `accel`.

## Pipeline steps

1. Loaded every selected raw subject file.
2. Standardized column names to `subject_id`, `activity_label`, `timestamp_ns`, `x`, `y`, `z` and added `device`, `sensor`, `source_file`, and `activity_name`.
3. Converted subject IDs and timestamps to integer nullable types, and sensor axes to numeric floats.
4. Normalized activity labels to uppercase and mapped labels to activity names.
5. Dropped rows with missing required fields, unknown labels, nonpositive timestamps, or subject/file mismatches.
6. Removed exact duplicate samples.
7. Sorted the cleaned data by subject, activity, and timestamp.
