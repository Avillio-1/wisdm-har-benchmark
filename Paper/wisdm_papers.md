| [Kwapisz, Weiss, and Moore, *Activity Recognition using Cell Phone Accelerometers*](https://www.cis.fordham.edu/wisdm/public_files/sensorKDD-2010.pdf) | 2011 | Original WISDM phone-accelerometer benchmark | 6 activities: walking, jogging, walking upstairs, walking downstairs, sitting, standing; 29 users | Android phone triaxial accelerometer, about 20 Hz | 10-fold cross-validation over examples/windows as reported in the original benchmark literature | No | J48 decision tree, logistic regression, multilayer perceptron | Handcrafted time-domain window features | Best reported accuracy about 91.7% with multilayer perceptron | Easier than unseen-subject evaluation and phone-accelerometer only | A scoped 6-class task is historically standard, but modern comparisons should split by subject. |
| [Weiss et al., *WISDM Smartphone and Smartwatch Activity and Biometrics Dataset*](https://archive.ics.uci.edu/dataset/507/wisdm+smartphone+and+smartwatch+activity+and+biometrics+dataset) | 2019 | Core WISDM 2019 dataset descriptor | 18 activities from 51 subjects | Phone and smartwatch accelerometer plus gyroscope, 20 Hz | Dataset descriptor; no single canonical split | N/A | Dataset release | Raw time-series plus transformed 10-second examples | No single canonical benchmark metric | Papers choose different subsets, sensors, windows, and splits | Use this source to justify a priori 18-class and sensor-ablation definitions. |
| [Peppas et al., *A Real-Time Activity Recognition Framework Using Smartphone Accelerometers*](https://www.mdpi.com/2076-3417/10/24/9040) | 2020 | WISDM phone-accelerometer HAR benchmark | Classic WISDM activities such as walking, jogging, stairs, sitting, and standing | Smartphone accelerometer only | 10-fold user-independent evaluation | Yes | CNN-style real-time framework with CNN/LSTM/BiLSTM comparators | Forty handcrafted time-domain accelerometer features | WISDM accuracy 95.27% | Phone-only and scoped task | A compact model with strong features can be competitive under user-independent evaluation. |
| [Oluwalade et al., *Human Activity Recognition using Deep Learning Models on Smartphones and Smartwatches Sensor Data*](https://www.scitepress.org/Papers/2021/104675/104675.pdf) | 2021 | WISDM 2019 smartphone/watch benchmark | 15-class task after dropping Eating Pasta, Eating Soup, and Kicking a Soccer Ball | Watch accel, phone accel, phone accel+gyro, watch accel+gyro; 5-second 100-step windows | Random 0.8/0.1/0.1 train/validation/test split | No | CNN, ConvLSTM, LSTM, BiLSTM | Raw sequence windows, 100x3 or 100x6 | Watch accel CNN about 94.47% accuracy; ConvLSTM 94.29%; BiLSTM 93.49%; LSTM 91.42% | Random split can mix subjects and nearby windows | Good inspiration for sensor ablations and deep baselines, but not a fair target score. |
| [Tan et al., *Human Activity Recognition Using a Hybrid Neural Network and a Regularized Extreme Learning Machine*](https://www.mdpi.com/1424-8220/23/18/7974) | 2023 | WISDM benchmark with windowed samples | 18 WISDM activities, 100-row windows, mixed-activity windows removed | Three-axis acceleration windows emphasized | Train/test split after window construction, 34,316 train and 14,706 test samples | No evidence | CNN-BiGRU-SE-RELM and recurrent/CNN ablations | Raw 100x3 windows | Accuracy 98.46%, F1 98.45% | High score is not protected against subject leakage | Removing mixed-activity windows is valuable; the reported score should not be compared to subject-wise results. |
| [Alexan et al., *Human Activity Recognition Using Machine Learning and Wearable Sensors*](https://www.mdpi.com/2076-3417/14/19/8846) | 2024 | WISDM 2019 subset study | 7 activities from the first five users: walking, jogging, stairs, sitting, standing, brushing teeth, kicking a soccer ball | Smartwatch accelerometer and gyroscope | Split not presented as subject-wise; first-five-user subset | No | Random forest, SVM, kNN, CNN, VGG16, MobileNetV2, LSTM | Sensor traces converted to 2D plot/image representations | RF 91.18% accuracy and 91.26% F1; CNN 89.42%; LSTM 83.18% | Small selected user subset and unusual image conversion | Classical random forest remains competitive; broad generalization claims need caution. |
| [Turdalyuly et al., *Wearable Sensor-Free Adult Physical Activity Monitoring Using Smartphone IMU Signals and Deep Learning*](https://www.mdpi.com/2078-2489/17/4/148) | 2026 | Recent WISDM 2019 cross-subject study | 6 broad a priori categories: stationary, walking, stairs, sport, household, eating | Phone accel, phone gyro, watch accel, watch gyro, and fusion setups | 5-fold GroupKFold cross-subject evaluation | Yes | CNN1D, LSTM, GRU, BiLSTM, fusion neural models | Raw IMU sequences | Best macro F1 about 46.0%; phone accel+gyro about 45.2%; phone+watch fusion about 43.4% | Broad remapping differs from the original 18-class task | Closest fairness match: subject-wise WISDM is much harder, and fusion is not automatically beneficial. |
